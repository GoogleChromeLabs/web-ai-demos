🛡️ How It Works

1

You type a message in the support chat.

2

Chrome's built-in Gemini Nano scans the text locally — no data leaves your device.

3

If sensitive data is detected, the message is blocked with a specific warning. Otherwise, it's "sent" normally.

Detects:

  • 💳 Credit / debit card numbers
  • 🔑 Passwords, PINs, security codes
  • 🆔 Social Security / national ID numbers
  • 🔐 API keys and access tokens
  • 🗝️ Private keys and certificates
  • 🏦 Bank account / routing numbers
  • 🛂 Passport numbers
  • 📱 Phone numbers
  • 📧 Email + password combos
Requirements: Chrome 148+ with the Prompt API enabled.
